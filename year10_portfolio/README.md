# About this project

## Back Story
During the summer holidays as a 14 year old student I took the initiative to self teach myself HTML, CSS and Javascript for web-design. I watched many tutorials and completed css challenges in the initial stages of my learning. I then went on to follow tutorial project website to understand both file structure and common practices. I then created this portfolio website for myself using my new found skills.

Unfortunately my love for programming was put on halt during school time as I had to study for my GCSEs however when it came to A-level choices I knew that I wanted to take computer science because I rememberd vividly how much I adored this initial project. I also, inbetween my A-level choices and this project, exposed myself to databases such as MySQL and MongoDB and backend programming through PHP and NODE.JS which only further encouraged my love and passion.
